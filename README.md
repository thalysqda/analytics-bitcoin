## 🎯 Sobre o Projeto
Este projeto consiste em uma **Análise Exploratória de Dados (EDA)** detalhada focada no histórico do Bitcoin (BTC), a principal criptomoeda do mercado financeiro global. 

O objetivo principal é extrair, limpar e analisar dados históricos de preço e volume, aplicando técnicas de análise de dados com Python para compreender ciclos de mercado, níveis de volatilidade e tendências de longo e curto prazo.

## 🔍 Objetivos da Análise e KPIs
A análise foi estruturada para responder a perguntas críticas do mercado de criptoativos, incluindo:
- **Evolução de Preço:** Como o preço de fechamento (Close) evoluiu ao longo dos anos?
- **Volatilidade e Risco:** Qual é a variação percentual diária do ativo e como os picos de volatilidade se correlacionam com eventos de mercado?
- **Indicadores de Tendência:** Aplicação de Médias Móveis Simples (SMA) para identificar tendências de alta (Bull Market) ou baixa (Bear Market).
- **Análise de Volume:** Qual a relação entre picos de volume de negociação e grandes movimentações de preço?

## 🛠️ Tecnologias Utilizadas
- **Linguagem:** Python
- **Manipulação e Limpeza de Dados:** Pandas, NumPy
- **Visualização de Dados:** Matplotlib, Seaborn (ou Plotly para gráficos interativos)
